# OpenZeppelin Skills

Agent skills for secure smart contract development with OpenZeppelin Contracts.

## Available Skills

| Skill | Purpose | Status |
|-------|---------|--------|
| [openzeppelin-contracts](skills/openzeppelin-contracts/) | Setup, patterns, and upgrades for OpenZeppelin Contracts | ✅ Draft for patterns, ❌ Placeholders for setup/upgrades |
| [openzeppelin-security](skills/openzeppelin-security/) | Secure development best practices | ❌ Placeholders |

## Installation

Copy the skill folder(s) to your `.claude/skills/` directory:

```bash
cp -r skills/openzeppelin-contracts ~/.claude/skills/
cp -r skills/openzeppelin-security ~/.claude/skills/
```

## Ecosystem-Specific Content

Skills may include general guidance and workflows, and optionally ecosystem-specific content in the `references/` directory of each skill.
