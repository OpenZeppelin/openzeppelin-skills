# OpenZeppelin Skills

Skills for secure smart contract development with OpenZeppelin Contracts.

## Available Skills

| Skill | Purpose | Status |
|-------|---------|--------|
| `openzeppelin` | General guidance and skill discovery | ✅ Initial draft |
| `openzeppelin-setup` | Project installation and configuration | ❌ Placeholders |
| `openzeppelin-patterns` | Pattern discovery via OpenZeppelin MCP generators | ✅ Initial draft |
| `openzeppelin-upgrades` | Upgradeable contract patterns | ❌ Placeholders |
| `openzeppelin-security` | Secure development best practices | ❌ Placeholders |

## Ecosystem-Specific Content

The `setup`, `patterns`, and `upgrades` skills include guidance for:

- **Solidity/EVM** — Hardhat, Foundry
- **Cairo/Starknet** — Scarb
- **Stylus** — Cargo + Stylus SDK
- **Stellar** — Soroban

## Security Best Practices

The `openzeppelin-security` skill covers security principles that apply to all smart contract development, with ecosystem-specific guidance in reference files.
